# FDS 산출물 보관·보존 (초안)

운영 전 합의용 템플릿이다. 실제 보존 기간은 조직 정책·개인정보 규정에 따른다.

| 구분 | 예시 경로 | 보존 제안 | 비고 |
|------|-----------|-----------|------|
| 지표·요약 JSON | `outputs/fds/*.json` | 90일~1년 | Git에 올릴 수 있는 작은 파일 |
| 검토 큐 CSV | `review_queue_*.csv` | 30~90일 | PII 가능 시 암호화·최소 컬럼 |
| 전체 결정 CSV | `decisions_*.csv` | 14~30일 또는 집계 후 삭제 | 용량·민감도 높음 |
| 모델 번들 | `*.joblib` | 모델 버전 관리 정책에 따름 | Git 미포함 |
| 로그 | 스케줄러 출력 리다이렉트 | 30일 | 실패 알림과 함께 |

**재현**: 소스·YAML·스크립트는 Git에 두고, 대용량 산출은 `pipeline_open_standard.py`로 재생성한다.
